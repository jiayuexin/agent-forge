## 3.3 预置模板库（初始版本）

| 模板 ID | 名称 | 适用场景 | 内置工具 |
|---|---|---|---|
| `customer-service` | 客服专员 | 用户咨询、投诉处理 | 订单查询、退款、通知 |
| `sales-assistant` | 销售顾问 | 产品推荐、报价生成 | 产品搜索、报价生成 |
| `code-reviewer` | 代码审查员 | 代码质量审查 | 文件读取、Lint 执行 |
| `content-writer` | 内容创作者 | 文案撰写、翻译 | 无（纯文本） |
| `data-analyst` | 数据分析师 | 数据查询、报表分析 | 数据库查询、图表生成 |
| `general` | 通用 Agent | 未匹配其他模板时 | 无 |

## 3.4 生成物目录结构

生成的 Agent 是一个完整的 npm 包，同时包含**业务 Agent 代码**和**客户端运行时**：

```
./agents/<name>/
├── src/
│   ├── index.ts          # Agent 类入口（导出 IAgent 实现）
│   ├── prompts.ts        # 系统 Prompt 模板
│   ├── tools.ts          # 工具定义
│   ├── types.ts          # 本地类型定义
│   └── runtime.ts        # AgentRuntimeClient（连接云端 Dashboard）
├── package.json
├── tsconfig.json
├── README.md
└── .agentforge.json      # Agent 元数据 + Dashboard 端点配置
```

### 新增文件说明

| 文件 | 说明 |
|---|---|
| `src/runtime.ts` | 轻量级运行时客户端，启动后连接 Dashboard，注册为 Agent Node，监听远程任务 |
| `.agentforge.json` | 除原有元数据外，增加 `dashboardUrl`、`authToken`、`allowRemoteExecution` 等运行时配置 |

### runtime.ts 核心职责

```typescript
export class AgentRuntimeClient implements IAgentRuntimeClient {
  async start(): Promise<void> {
    // 1. 初始化 Agent
    // 2. 连接 Dashboard WebSocket
    // 3. 注册节点信息
    // 4. 启动心跳
    // 5. 监听 ControlMessage
  }

  async onControlMessage(message: ControlMessage): Promise<void> {
    if (message.type === 'execute') {
      const result = await this.agent.execute(message.payload.task);
      this.send({ type: 'result', messageId: message.messageId, nodeId: this.nodeId, payload: result });
    }
  }
}
```

## 3.5 生成后安全边界

生成引擎输出的是**可执行代码**，必须明确其执行边界，防止 LLM 生成结果被直接滥用。

### 默认行为

- 生成产物**不自动执行**，仅写入输出目录。
- 生成完成后返回 `GenerateResult`，包含文件列表、元数据和风险等级。

### 自动执行前提

仅当同时满足以下条件时，才允许通过 `--execute` 等显式选项直接运行生成的 Agent：

1. TypeScript 编译检查通过。
2. 模板风险等级为 `low`（如 `content-writer`、`general`）。
3. 生成结果未包含标记为 `sensitiveOperations` 的能力（如退款、签约、删改数据）。
4. 用户显式传入 `--execute` 或等效确认标志。

### 人工 review 要求

包含以下任一因素的生成结果，必须经人工 review 或显式 `--confirm-execute` 后才可运行：

- 涉及资金、退款、合同、权限变更等敏感操作。
- 模板风险等级为 `medium` 或 `high`。
- 生成过程中工具推荐命中了不在模板预置列表中的工具。

### 禁止行为

生成引擎及 CLI 不得：

- 在未经确认的情况下自动安装依赖或执行 shell 命令。
- 远程加载或热更新 Agent 代码。
- 将用户环境变量或 API Key 写入生成文件（`.agentforge.json` 仅允许保存 Dashboard 端点等非敏感配置）。
